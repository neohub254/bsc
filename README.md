## 🚀 About The Project

CIVIL-X OS is a futuristic, cyberpunk-themed educational website for BSC Civil Engineering students. It combines engineering tools with Gen Z humor to create an engaging learning platform.

**"Netflix + Cyberpunk + Civil Engineering + Gen Z Memes"**

---

## ✨ Features

- 🎨 **Futuristic UI** - Glassmorphism, neon effects, blueprint grids
- 📚 **Subject Dashboard** - Browse all 8 engineering modules
- 📝 **Resource Management** - Notes, past papers, YouTube tutorials
- 💬 **Community Hub** - Announcements, memes, confessions
- ⏰ **Exam Countdown** - Real-time countdown to exams
- 📊 **Stress Calculator** - Because engineering is hard
- 🎭 **Meme Generator** - Engineering humor at its finest
- 📩 **Complaint System** - Direct email to university
- 📱 **Fully Responsive** - Works on all devices
- 🌙 **Dark Mode Only** - As all engineering websites should be

---

## 🛠️ Technologies Used

- **HTML5** - Structure
- **CSS3** - Styling with animations
- **Vanilla JavaScript** - All functionality
- **No Frameworks** - Pure code only
- **No Backend** - Everything runs client-side
